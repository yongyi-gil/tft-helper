## Getting started
1. 프로젝트 인스톨
```bash
yarn instal
```

2. env.local 파일 생성
```
root에 있는 env.local.sample 파일 복사 후 env.local 파일 생성
```

3. TFT Developer key 발급 후 env.local에 입력
> 키 발급 : https://developer.riotgames.com

4. 프로젝트 실행
```bash
yarn dev
```

4. json-server 실행
```
npx json-server --port 9999 -watch db.json
```
